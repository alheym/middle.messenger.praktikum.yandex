## Описание

Учебный проект по курсу Яндекс.Практикум - чат без использования фреймворков

Использовались:

- Сборщик Parcel
- Шаблонизатор Handlebars
- Netlify

## Установка

- `npm run dev`
- `npm run start`
- `npm run build`
